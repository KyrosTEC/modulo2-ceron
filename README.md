# Computer Vision for Robotics Module 👁️🤖

## Overview

Repository for activities in the **Computer Vision for Robotics module**, focused on building perception pipelines for robotic systems using camera data.

## Topics

- Image capture
- Image processing
- Segmentation
- Feature extraction
- Object classification
- OpenCV-based video processing

## Activity: Traffic Sign Detection 🚦

The main activity consists of detecting and classifying traffic signs such as:

- Stop
- Workers
- Go Straight
- Turn Left / Right

The pipeline combines techniques such as:

- Image quality metrics
- Template Matching
- Haar Cascade
- CNN-based detection, such as YOLO

Testing is performed using the **3DGS simulator** or a real robotic track.
